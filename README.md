## Installation

### Prerequisites

- [Node.js 18+](https://nodejs.org)
- [pnpm](https://pnpm.io)

### Setup

```bash
# Clone the repository
git clone https://github.com/MainakVerse/Medix.git
cd Medix

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

### Build for Production

```bash
pnpm build
```
